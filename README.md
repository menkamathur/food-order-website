# 🍔 Food Order Website

A simple and responsive **Food Ordering Website** built using **HTML, CSS, and JavaScript**.
The website allows users to browse food items, search meals, manage cart items, and navigate through different pages such as login, profile, and cart.

---

## 🚀 Features

* 🏠 Home page with food categories
* 🔍 Search food items
* 🛒 Cart page for selected items
* 👤 Login page UI
* 📄 Profile page layout
* 📦 Food item listing
* 📱 Responsive design
* 🎨 Clean UI using CSS

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript**
* **VS Code**

## ▶️ How to Run the Project

1. Clone the repository

```
git clone https://github.com/menkamathur/food-order-website.git
```

2. Open the project folder

```
cd food-order-website
```

3. Open `index.html` in your browser.

Or run using **Live Server** in VS Code.
